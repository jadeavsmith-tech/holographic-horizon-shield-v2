# Security Policy: Holographic Horizon Shield V2 (HHS-V2)

## Supported Versions
Only the latest commit on the primary development branch is actively supported.

| Version | Supported          |
| ------- | ------------------ |
| v2.x    | :white_check_mark: |
| v1.x    | :x:                |

## Reporting a Vulnerability
As an independent researcher, I am dedicated to maintaining a secure perimeter. If you discover a security vulnerability, algorithmic bypass, or edge proxy exploit within HHS-V2, please **do not open a public GitHub issue.** 

Instead, report the vulnerability responsibly via the following channel:
* **Email:** [Your Professional Project Email Here]

### What to Include in Your Report:
1. **Payload Trajectory:** The exact token distribution, sequence, or structural payload used to execute the bypass.
2. **Environment Details:** The proxy configuration and downstream LLM model context.
3. **Proof of Concept:** A minimal, reproducible script or command sequence.

### Disclosure Timeline
I will acknowledge receipt of your report within 48 hours and work toward a deterministic patch. Public disclosure will be coordinated after a mitigation vector is pushed to the edge kernel.
